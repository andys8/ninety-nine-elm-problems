# Problem 19 Solution

```elm 
rotate : Int -> List a -> List a
rotate rot list =
    let
        r =
            rot % (List.length list)
    in
        if r >= 0 then
            List.drop r list ++ List.take r list
        else
            List.take (abs r) list ++ List.drop (abs r) list
```
