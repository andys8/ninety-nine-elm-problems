# Problem 15 Solutions

## Solution 1 - recurse
```
repeatElements : Int -> List a -> List a
repeatElements n list =
    case list of 
        [] -> []
        
        x :: xs -> 
            (List.repeat n x) ++ repeatElements xs n
```

## Solution 2 - concatMap
```elm
repeatElements : Int -> List a -> List a
repeatElements n list =
    List.concatMap (List.repeat n) list
```
