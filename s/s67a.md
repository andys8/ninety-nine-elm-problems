#Problem 67a Solutions

##Solution 1

toString on String and C

```elm
treeToString : Tree a -> String
treeToString tree =
    case tree of
        Empty ->
            ""
          
        Node v Empty Empty ->
            String.filter noQuotes (toString v) 

        Node v left right ->
            String.filter noQuotes
              ((toString v) 
                ++ "(" 
                ++ treeToString left
                ++ ","
                ++ treeToString right
                ++ ")")



noQuotes : Char -> Bool
noQuotes c = 
    (c /= '\'') && (c /= '\"')
```
