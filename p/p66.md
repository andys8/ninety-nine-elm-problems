# Problem 66

The illustration below depicts yet another layout strategy. 

![](../i/p66.gif)
The method yields a very compact layout while maintaining a certain symmetry in every node. Find out the rules and write the layout function.

layout tree65 ==
    Node ('n',(5,1)) 
        (Node ('k',(3,2)) 
            (Node ('c',(2,3)) ...
