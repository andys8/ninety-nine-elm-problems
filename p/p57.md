# Problem 57

Build a binary search tree from a list. Lower values should be placed to the right

##Example 
```elm
toTree [1, 2, 3, 0] == Node 1 
                  (Node 2 Empty (Node 3 Empty Empty)) 
                  (Node 0 Empty Empty)
```
