# Problem 23

Extract a given number of randomly selected elements from a list.

```
randomSelect 3 ["Al", "Biff", "Cal", "Dee", "Ed", "Flip"]
```

Example result:

```
randomSelect 3 ["Al", "Biff", "Cal", "Dee", "Ed", "Flip"] == ["Cal", "Dee", "Al"]
```
